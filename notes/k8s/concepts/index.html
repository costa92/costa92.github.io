Kubernetes 概念和术语

Kubernetes（简称 K8s）是一个开源的容器编排平台，用于自动化容器化应用的部署、扩缩和管理。

Pod

Pod 是 Kubernetes 中最小的可部署单元。一个 Pod 可以包含一个或多个容器，这些容器共享存储和网络资源。

apiVersion: v1
kind: Pod
metadata:
  name: nginx-pod
spec:
  containers:
  - name: nginx
    image: nginx:1.21
    ports:
    - containerPort: 80

Service

Service 是一种将运行在一组 Pod 上的应用暴露为网络服务的抽象方式。

  • ClusterIP：集群内部访问（默认） - NodePort：通过节点端口对外暴露 - LoadBalancer：使用云提供商的负载均衡器对外暴露

Deployment

Deployment 为 Pod 和 ReplicaSet 提供了声明式更新能力，支持滚动更新和回滚。

apiVersion: apps/v1
kind: Deployment
metadata:
  name: nginx-deployment
spec:
  replicas: 3
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx
        image: nginx:1.21

Namespace

Namespace 提供了一种在集群内隔离资源组的机制，适用于多团队或多项目的环境。

ConfigMap 和 Secret

  • ConfigMap：用于存储非敏感的配置数据 - Secret：用于存储敏感数据（如密码、Token）
